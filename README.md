# 📱 WhatsApp Civil Helpdesk

A WhatsApp-based helpdesk system that allows citizens to register complaints related to civic issues such as road damage, water supply, electricity problems, garbage collection, and more.
The system automates complaint registration, tracking, and routing to the appropriate government departments.

---

# 🚀 Features

## 1. Complaint Registration

Citizens can register complaints directly through WhatsApp.
The bot collects essential information such as:

* Name
* Location
* Complaint category
* Issue description

Each complaint receives a **unique Complaint ID** for tracking.

---

## 2. Complaint Categories

Users can select the type of issue to ensure proper routing.

Examples:

* Road Damage
* Water Supply Issues
* Electricity Problems
* Garbage Collection
* Public Health Issues
* Government Office Complaints

---

## 3. Media Upload Support

Users can attach supporting media to complaints:

* 📷 Photos
* 🎥 Videos
* 📍 Location

This helps authorities understand the issue more clearly.

---

## 4. Complaint Status Tracking

Citizens can check the status of their complaint anytime using their **Complaint ID**.

Example statuses:

* Submitted
* In Progress
* Resolved

---

## 5. Automatic Department Routing

The system automatically routes complaints to the relevant government department based on the selected category.

Example:

* Road Issues → Municipal Department
* Electricity Problems → Electricity Board
* Water Supply Issues → Water Department

---

## 6. Admin Dashboard

Government staff can manage complaints using an admin dashboard.

Admin capabilities include:

* Viewing all complaints
* Filtering complaints by category or location
* Assigning complaints to employees
* Updating complaint status
* Monitoring resolution progress

---

## 7. Notifications & Updates

Users receive automatic WhatsApp updates when:

* Complaint is received
* Complaint is assigned
* Work has started
* Issue has been resolved

---

## 8. Multi-Language Support

The helpdesk supports multiple languages for accessibility.

Examples:

* English
* Hindi

---

## 9. FAQ & Government Information

Citizens can also get answers to common questions such as:

* Office timings
* Required documents
* Public service information
* Government helpline numbers

---

## 10. Feedback System

After a complaint is resolved, citizens can provide feedback and rate the service to help improve the system.

---

* **Frontend:** React / Next.js
* **Backend:** Node.js / Express
* **Database:** MongoDB / PostgreSQL
* **Messaging API:** WhatsApp Business API
* **Hosting:** Vercel / AWS / Render

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/prakhar123561/civichelp-desk.git
```

Move into the project directory:

```bash
cd civichelp-desk
```

Install dependencies:

```bash
npm install
```

Start the frontend (Next.js):

```bash
npm start
```

Run backend API server:

```bash
npm run backend:start
```

For auto-reload during development:

```bash
npm run backend:dev
```
